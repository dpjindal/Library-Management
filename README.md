# Library-Management
Library Management System
Welcome to Library Management System using PHP & MySQL.

Features:
Add Book,
View Books (Add/Edit),
Find Book,
Issue Book,
Return Book,
Add Student,
Barcode System,

Open the Index.php in your local server. Before invoking any .php file please create a database and related tables from the .sql file.
It is well working project of Library Management.
In local server like xampp, create a folder for 'library_management' under htdocs and copy all files in this folder except .sql file.
In liver server, create a folder name 'library_management' in the root folder of our hosting site and copy all files there and run the site as per folder name 'library_management' means write library_management in the browser address bar and your fullfledged site is ready to run.
